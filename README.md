### Dataset

### Objective

### Tools & Libraries
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

### Key Results
- Built and evaluated a Decision Tree model
- Generated classification report and confusion matrix
- Visualized model decision boundaries
